# Učimo NodeJS

## Instalacija

Kloniraj projekat:

```
git clone https://github.com/skolakoda/ucimo-nodejs.git
```

Nakon kloniranja, za svaki nivo je potrebno ažurirati zavisnosti:

```
npm install
```

## Nivoi

* Nivo 1: Pravimo server
* Nivo 2: Čitamo API
* Nivo 3: Završna aplikacija
